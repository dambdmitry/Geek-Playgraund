# Название Игроков
Geek Playground - это система, предоставляющая возможнность реализовать свой программерский потенциал путем решения игровых задач и играя с другими игроками. Учавствуй в турнирах, занимай призовые места и будешь увековечен на доске почета справа. В системе имеется определенный набор языков программирования. Подробнее о наборе можно посмотреть на странице "Игрокам".

Также имеется возможность создавать свои турниры. Это актуально для преподавателей, чтобы проводить игры между студентами. Это позволяет внести состязательные и игровые элементы в процесс обучения. Подробнее об организации турниров можно посмотреть на странице "Организаторам".

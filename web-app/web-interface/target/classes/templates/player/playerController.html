Как стать игроком турнира?

Для того, чтобы принять участие в турнире требуется выбрать турнир со статусом "OPEN" и нажать на кнопку "Стать учасстником турнира". Если турнир приватный, то нужно будет ввести секретный ключ.

Как отправлять решения?

После успешной регистрации на турнир на странице турнира появится кнопка "Отправить решение". Далее будет переход на страницу с правилами игры турнира и окно для ввода кода программы. Также есть список с выбором языка программирования. Для каждого языка подготовлен шаблон кода, которого нужно придерживаться. Рекомендуется писать и отлаживать свой код в вашей любимой IDE. При отправке ваш код тестируется системой. Если он содержит синтаксические или логические ошибки в контексте игры, то ошибка выведется на страницу и будет предложено отправить решение повторно, в ином случае вы увидите надпись, что все успешно и ваше решение сохранено. Далее нужно только ожидать начала турнира.

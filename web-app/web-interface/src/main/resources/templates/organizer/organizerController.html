Кто такой Организатор?

Организатор - это почетная роль системы Geek Playground. Являясь организатором появляется возможность создавать собственные турниры, такая роль приемлема для преподавателей, кто учит студентов программированию.

Организатор может создавать приватные турниры с паролем, чтобы органичить набор игроков. Также он имеет право участвовать в других турнирах и в своих.
После того, как в турнире наберется более одного принятого решения - появится кнопка для запуска турнира. По мере отправления решений игроками будет наполняться таблица иллюстрирующая имя игрока, язык и код программы.

Когда турнир начнется откроется таблица игроков. Она будет постепенно обновляться по завершению игр между участниками. После того как турнир завершиться появится надпись, о том, что турнир завершен. И можно уже анализировать результаты игроков. Также по нажатию на строку таблицы игроков можно посмотреть результаты отдельного игрока в виде таблицы, кого он победил, а кому проиграл. В этой таблице тоже можно нажать на строку и перейти в подробный отчет по игре.

Как создавать турниры?

Чтобы создавать турниры - нужно получить роль Организатора. После получения роли на странице "Турниры" появится кнопка "Создать турнир". Будет предложено ввести название турнира, пароль для создания приватного турнира, его можно оставить пустым и тогда турнир будет публичным, и выбрать игру для турнира.

Как стать Организатором?

Чтобы стать Организатором, нужно отправить заявку, которую одобряет или откланяет администратор. После одобрения заявки у вас появится кнопка "Создать турнир".
